# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2025-01-01

### Added
- Initial open-source release
- Django REST API backend with PostgreSQL
- Express.js high-performance API
- Next.js 15 frontend with admin panel
- Smart TV application (Samsung Tizen, LG webOS, Android TV)
- Mobile application (Expo — iOS and Android)
- HLS proxy service (Go)
- MediaMTX integration for live streaming (RTSP, RTMP, HLS, WebRTC)
- SeaweedFS S3-compatible storage
- Caddy reverse proxy with automatic SSL
- Schedule and playlist automation tools
- Video transcoding and normalization utilities
- Multilingual support (Kazakh, Russian, English)
- Docker Compose production deployment
- CI/CD pipeline with GitHub Actions
- Comprehensive documentation
