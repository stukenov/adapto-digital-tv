# iOS-Style Design System for Adapto Digital TV

## Обзор

Современная дизайн-система в стиле iOS для универсального Smart TV приложения Adapto Digital TV. Основана на
принципах дизайна Apple с адаптацией под телевизионные экраны.

## Основные принципы

- **Минимализм**: Чистые линии, функциональность превыше всего
- **Glassmorphism**: Эффекты размытия и полупрозрачности
- **Градиенты**: Тонкие градиенты для создания глубины
- **Анимации**: Плавные переходы с использованием iOS easing
- **Типографика**: Системные шрифты Apple для отличной читаемости

## Цветовая палитра

### Основные цвета

- **Primary Blue**: `#007AFF` - основной акцентный цвет iOS
- **Secondary**: `#5856D6` - дополнительный акцент (индиго)

### Темная тема

- **System Background**: `#000000` - основной фон
- **Primary**: `#1c1c1e` - первичный контейнер
- **Secondary**: `#2c2c2e` - вторичный контейнер
- **Elevated**: `rgba(58, 58, 60, 0.6)` - приподнятые поверхности

### Семантические цвета

- **Success**: `#34C759` - iOS зеленый
- **Warning**: `#FF9500` - iOS оранжевый
- **Error**: `#FF3B30` - iOS красный

## Компоненты

### Button

```tsx
<Button variant='primary' size='large' focused>
  Основная кнопка
</Button>
```

**Варианты**: `primary`, `secondary`, `tertiary`, `destructive` **Размеры**: `small`, `medium`,
`large`

### Card

```tsx
<Card variant='elevated' interactive>
  Контент карточки
</Card>
```

**Варианты**: `default`, `elevated`, `glass`, `highlight`

### Badge

```tsx
<Badge variant='live' pulse>
  В эфире
</Badge>
```

**Варианты**: `default`, `success`, `warning`, `error`, `info`, `live`

### Grid

```tsx
<Grid minItemWidth='300px' gap='lg' responsive>
  {items.map(item => (
    <Item key={item.id} />
  ))}
</Grid>
```

## Анимации

### Easing Functions

- **Spring**: `cubic-bezier(0.175, 0.885, 0.32, 1.275)` - iOS пружинящая анимация
- **Ease Out**: `cubic-bezier(0, 0, 0.2, 1)` - стандартное замедление

### Длительность

- **Fast**: `150ms` - быстрые переходы
- **Normal**: `300ms` - стандартные переходы
- **Slow**: `500ms` - медленные переходы

## Эффекты

### Glassmorphism

```css
backdrop-filter: blur(20px);
-webkit-backdrop-filter: blur(20px);
background: linear-gradient(135deg, rgba(255, 255, 255, 0.08) 0%, rgba(255, 255, 255, 0.02) 100%);
```

### Тени

- **Medium**: Стандартные тени для карточек
- **Large**: Выраженные тени для модальных окон

## Адаптивность

### Точки перелома

- **Mobile**: `< 768px` - одноколоночная сетка
- **Tablet**: `768px - 1200px` - двухколоночная сетка
- **Desktop**: `> 1200px` - многоколоночная сетка
- **4K TV**: `> 3840px` - увеличенные шрифты

### Стратегия масштабирования

1. **1080p**: Базовый размер шрифта `16px`
2. **1440p**: Увеличение до `18px`
3. **4K**: Увеличение до `24px`

## Доступность

- **Focus States**: Четкие индикаторы фокуса для TV навигации
- **Контрастность**: Соответствие WCAG AA
- **Уменьшенная анимация**: Поддержка `prefers-reduced-motion`
- **Высокий контраст**: Поддержка `prefers-contrast: high`

## Производительность

- **Hardware Acceleration**: Использование `transform` вместо изменения position
- **Избегание Layout Thrash**: Анимации только `transform` и `opacity`
- **Оптимизация FPS**: Цель 60 FPS на всех устройствах
