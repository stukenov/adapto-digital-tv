# Быстрая настройка Adapto Digital TV API

## Шаги для интеграции с реальным API

### 1. Настройка бэкенда
Убедитесь, что ваш Adapto Digital TV API сервер запущен на `http://127.0.0.1:8000`

### 2. Добавление учетных данных

Откройте файл `services/adapto-api.ts` и добавьте ваши учетные данные:

```typescript
// Замените пустые строки на ваши данные
const API_USERNAME = 'your_username_here';
const API_PASSWORD = 'your_password_here';
```

### 3. Проверка подключения

Запустите приложение:
```bash
npm start
```

### 4. Тестирование

1. Откройте главный экран - должны загрузиться реальные каналы
2. Перейдите на экран канала - должно отобразиться расписание
3. Проверьте экран "Расписание" - должны показаться программы по дням

### 5. Устранение проблем

**Ошибка подключения к API:**
- Проверьте, что сервер запущен на правильном порту
- Убедитесь, что API доступно по адресу `/api/v1`
- Проверьте учетные данные

**Пустой список каналов:**
- Убедитесь, что в базе данных есть каналы с `is_active = true`
- Проверьте корректность API endpoint'ов

**Проблемы с CORS:**
- Настройте CORS на бэкенде для локального development
- Добавьте `http://localhost:*` в allowed origins

### 6. Логирование

Откройте Developer Tools в браузере/эмуляторе для просмотра логов API запросов.

## Готово!

После выполнения этих шагов ваше приложение будет работать с реальным API Adapto Digital TV вместо моковых данных.
